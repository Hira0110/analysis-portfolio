# データ分析ポートフォリオ

このリポジトリでは、Python（pandas / matplotlib / seaborn）を用いた  
データ前処理・可視化・簡易的な統計分析の実例をまとめています。

---

## 📁 プロジェクト構成

analysis_project  
├─ data/　　　　… 使用データ（sales.csv）  
├─ notebook/　… Jupyter Notebook（portfolio_analysis.ipynb）  
└─ output/　　 … 生成されたグラフ画像  

---

## 📘 使用技術

- Python 3.x  
- pandas  
- matplotlib  
- seaborn  
- Jupyter Notebook（VS Code）

---

## 📊 分析内容

### 1. データ前処理
- 欠損値確認  
- 日付型への変換  
- 月別売上の集計（groupby）  

### 2. 可視化
- 月別売上推移（折れ線グラフ）  
- 広告費と売上の関係（散布図＋回帰直線）  

### 3. 簡易分析
- 広告費と売上の相関  
- カテゴリ別の売上傾向
